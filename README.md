# Anul Recommends

Simple affiliate site for Amazon Associates.

## Deploy to GitHub Pages
1. Go to Settings → Pages → Source → `Deploy from branch`.
2. Choose branch: `main`, folder: `/ (root)`.
3. Save — your site will be live at:  
   https://anul011.github.io/an/

## Customize
- Replace YOURTAG-20 with your Amazon Associates tag.
- Update email and image links.
- Commit and push changes — Pages updates automatically.
